La cartella schermate.zip contiene la mappa globale del dataset01 e i vari campioni di mappa virtuale impostando x a diversi valori (in ordine: 10, 50, 100, 150, 120).
La cartella robotica.zip contiene una sottocartella denominata progettopcl all'interno della quale si trovano le cartelle 
    rofl-main (serve per leggere i parametri a linea di comando) 
    build, il file main.cpp che si occupa dell'esecuzione del ray tracing.
